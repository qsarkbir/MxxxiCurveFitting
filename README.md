MxxxiCurveFitting

リオネル・メッシのワールドカップ「1試合あたり得点」に、いろいろな回帰モデルを当てはめて未来を「予測」する、インタラクティブなブラウザツールです。手描き風（Rough.js）の一枚絵で、外挿（extrapolation）と過剰適合（overfitting）の危うさを体感することを目的にしています。

元ネタ

このツールは xkcd のコミックが元ネタです。

出典：https://xkcd.com/3260 （xkcd #3260 "Messi"）

オリジナルのコミックは、メッシのワールドカップ平均得点という少数の点に指数曲線を当てはめ、「この調子なら2040年には1試合あたり数百ゴール」と外挿してみせる風刺画です。小さなサンプルサイズ（特に「2026（現時点）」が1試合だけの3ゴールに基づく点）が、外挿をいかに暴走させるかを示しています。


xkcd の作品は Creative Commons 表示-非営利 2.5 ライセンス の下で公開されています（共有は自由ですが販売は不可）。本リポジトリはコミックの画像を再配布していません。リンクのみで参照しています。



使い方

messi-curve-fitting.html をブラウザで開くだけです。ビルドもサーバも不要で、すべて1ファイルに収まっています。


モデルを選ぶ — 線形・多項式（2〜4次）・完全補間・指数・対数・べき乗・折れ線・平均（傾きなし）の10種類。
予測する大会を選ぶ — スライダーで 2030〜2050 年を指定。
2026年のジョーク点（3.0）の ON/OFF — たった1点が外挿をどれだけ歪めるかを切り替えて確認できます。


グラフの縦軸は予測値に合わせて自動でスケールし、左上には選択中モデルの当てはめ式が実際の係数つきで表示されます。

データ

横軸の t は大会番号（2006→1, 2010→2, … 2026→6）です。

2006〜2022 は実際の戦績です。2026 はコミックに合わせた「1試合3ゴール」のジョーク点で、本来こうした単一試合の値をそのまま平均扱いすべきではありません——その不適切さを誇張するために、あえて入れています。

教訓

R² が高い（データへの当てはまりが良い）モデルほど、外挿は派手に裏切ります。このツールで一番「まとも」な未来を返すのは、統計的には一番初歩的な 相加平均（傾きなしモデル） です。過去をうまく説明することと、観測範囲の外を正しく言い当てることは別物だ、というのが要点です。

技術メモ


依存ライブラリは Rough.js（手描き風描画）のみ。CDN から読み込みます。
フォントは Google Fonts の Patrick Hand / Yomogi。読み込めない環境では Comic Sans 系にフォールバックします。
回帰計算（最小二乗法、対数変換による指数・べき乗フィット等）はすべて素のJavaScriptで実装しています。
